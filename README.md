# Criz Clothing Website

A nature and seal-inspired clothing brand website built with HTML, CSS, and JavaScript.

## Description

Criz Clothing celebrates the beauty of nature and the grace of seals through sustainable fashion. This website showcases our collections and brand story.

## Features

- Responsive design
- Product catalog
- About page with brand story
- Contact form
- Nature and seal themed styling

## Structure

- `index.html` - Home page with hero section and featured products
- `products.html` - Product catalog page
- `about.html` - About the brand page
- `contact.html` - Contact form page
- `css/style.css` - Main stylesheet
- `js/script.js` - Basic JavaScript functionality
- `images/` - Directory for product and theme images

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the different pages using the menu

## Customization

- Replace placeholder images in the `images/` folder with actual product photos
- Update the color scheme in `css/style.css` to match your brand
- Add more products to `products.html`
- Implement backend functionality for the contact form and shopping cart

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
